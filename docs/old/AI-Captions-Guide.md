‚úÖ **DO:**
- Use a good quality microphone

- Minimize background noise
- Use proper grammar and pronunciation
- Test with a 10-second trial before going live

4. Test audio levels before enabling captions

- **International events:** Start with English, switch based on speaker
- **Language learning:** Show captions in target language for practice

## Troubleshooting

### Captions Not Appearing
**Problem:** Captions enabled but nothing shows
**Solutions:**
1. Check microphone permissions in browser
2. Verify microphone is selected in Settings
3. Ensure browser supports Web Speech API (Chrome, Edge recommended)
4. Test by speaking directly into mic
5. Try refreshing the page

### Inaccurate Transcription
**Problem:** Captions show wrong words
**Solutions:**
1. Improve microphone quality
2. Reduce background noise
3. Speak more clearly and slowly
4. Check if correct language is selected
5. Avoid technical jargon/slang

### Captions Lag Behind Speech
**Problem:** Delay between speaking and captions
**Solutions:**
1. This is normal (1-2 second delay for processing)
2. Close other browser tabs to free up resources
3. Use Chrome/Edge for best performance
4. Restart browser if lag exceeds 3 seconds

### Language Not Available
**Problem:** Desired language not in list
**Solutions:**
1. Check if browser supports the language
2. Use closest language variant (e.g., Spanish-US ‚Üí Spanish-Mexico)
3. Request language addition via feedback

## FAQ

**Q: Do captions work in Producer Mode?**
A: Yes, captions are enabled per-stream, not per-user.

**Q: Can viewers see captions?**
A: Yes, if you're streaming to platforms with caption support (YouTube, Facebook).

**Q: Can I edit captions in real-time?**
A: No, captions are auto-generated and cannot be edited live.

**Q: Do captions appear in recordings?**
A: Yes, if captions are visible on stream, they're recorded.

**Q: Can I customize caption appearance?**
A: Not currently (future feature).

**Q: Do captions work with music?**
A: No, captions only transcribe speech, not music lyrics.

**Q: Can multiple languages be shown simultaneously?**
A: No, only one language at a time.

**Q: Are captions saved for later?**
A: No, captions are live-only and not saved.
